# Ebenezer Concrete - Design System

## Color Palette

### Primary Colors
- **Blue:** #2196F3 (Hero sections, CTAs, accents)
- **Green:** #4CAF50 (Service icons, success states)
- **Gold/Yellow:** #FFD700 (Primary CTAs, highlights, footer headings)

### Neutral Colors
- **White:** #FFFFFF (Backgrounds, cards)
- **Dark Gray:** #263238 (Footer background)
- **Medium Gray:** #333333 (Headings)
- **Light Gray:** #666666 (Body text)
- **Very Light Gray:** #E0E0E0 (Borders, dividers)
- **Background Gray:** #F8F9FA (Page backgrounds)

## Typography

### Font Family
- Primary: Segoe UI
- Fallback: Tahoma, Geneva, Verdana, sans-serif

### Font Sizes
- **H1 (Hero):** 56px, Bold
- **H2 (Page Title):** 48px, Bold
- **H3 (Section Title):** 40px, Bold
- **H4 (Card Title):** 22-24px, Bold
- **Body Text:** 18px, Regular
- **Small Text:** 16px, Regular
- **Caption:** 14px, Regular

### Line Heights
- Headings: 1.2
- Body text: 1.6-1.8

## Spacing System

### Grid
- Base unit: 8px
- Common spacing: 20px, 40px, 60px, 80px

### Component Spacing
- Card padding: 30-40px
- Section padding: 60-80px
- Button padding: 15-20px (vertical), 30-50px (horizontal)

## Components

### Buttons
- **Primary CTA:** Yellow (#FFD700), 12px border radius
- **Secondary CTA:** Blue (#2196F3), 8px border radius
- **Height:** 40-60px (depending on importance)
- **Min width:** 160px

### Cards
- **Background:** White (#FFFFFF)
- **Border radius:** 12px
- **Shadow:** 0 4px 20px rgba(0,0,0,0.08)
- **Padding:** 30-40px
- **Hover:** Lift effect with increased shadow

### Icons
- **Size:** 60-70px diameter circles
- **Color:** Green (#4CAF50)
- **Background:** Solid fill
- **Icon:** Emoji or simple graphics

## Layout

### Desktop Breakpoints
- Max width: 1920px
- Content width: 1400px (max)
- Grid columns: 3-4 columns

### Header
- Height: 70px
- Background: White
- Shadow: 0 2px 8px rgba(0,0,0,0.1)

### Footer
- Background: #263238
- Padding: 40-60px
- 4-column layout

## Accessibility

### Color Contrast
- Text on white: #333333 (AAA compliant)
- Text on blue: #FFFFFF (AA compliant)
- Minimum font size: 14px

### Interactive Elements
- Minimum touch target: 44x44px
- Clear hover states
- Visible focus indicators

## Animation

### Transitions
- Duration: 0.3-0.4s
- Easing: Ease-out
- Type: Dissolve/fade

### Hover Effects
- Button: Slight shadow increase
- Card: Translate Y -5px + shadow
